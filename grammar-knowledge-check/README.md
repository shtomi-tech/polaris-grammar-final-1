# 英文法 基礎知識チェック

英文法の用語・形・働き・区別を確認する、60問・4択の静的アプリです。

## ねらい

- 用語を知っているかだけでなく、似た形を**識別**し、文脈から**適用**できるかを分けて確認する
- 誤答を「よくある混同」として記録し、分野別の詳しい解説につなげる
- 重要な12問では、答えに加えて根拠も選び、偶然の正答と理解を分ける
- 通常利用ではブラウザの localStorage に保存し、生徒別共有URLでは共通Supabaseへ結果を同期する

## 診断の見方

- **知識**: 用語・基本ルールを再生できるか
- **識別**: 似た形・意味を根拠を持って見分けられるか
- **適用**: 文脈から必要な形を判断できるか

分野判定は、1問の誤答だけで「未定着」とはしません。正答率、重要問題の正誤、保留を合わせて「定着／要確認／未定着」とし、2問以下の分野は「要追加確認」と表示します。

## 起動

```powershell
cd "C:\Users\shtom\dev\polaris-grammar-final-1\grammar-knowledge-check"
py -3 -m http.server 8097
```

`http://127.0.0.1:8097/` を開く。

## 構成

- `data/questions.js`: 17分野・60問、測定区分、誤概念タグ、重要問題の根拠選択、分野別解説
- `static/app.js`: 出題、根拠選択、採点、能力別・誤概念別の結果、解説、ローカル保存・生徒別クラウド同期
- `static/styles.css`: アイボリー地・影なし・キーボード操作前提の画面

## 検証

```powershell
node --check data/questions.js
node --check static/app.js
node scripts/check-data.js
```

`check-data.js` は問題数・選択肢・正解に加え、測定区分、誤概念タグ、根拠選択肢を検査します。否定表現や選択肢長の偏りは警告として出力します。
