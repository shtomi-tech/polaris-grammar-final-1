# 英文法 基礎知識チェック

初学者向けの入口です。5段階（品詞と文の骨組み／動詞の形と時制／助動詞・受動態と準動詞／準動詞と文のつなぎ方／比較・仮定法と仕上げ）に分けて進められます。入試形式の演習は、同じ公開サイトのPolaris入試基礎演習へ進みます。

英文法の用語・形・働きを網羅的に確認する、150問・4択の静的アプリです。1回30問を5段階で進めます。

原則集は問題範囲の全体を定めるものではありません。active原則は該当する問題の正誤判定・解説の基準として使い、原則集にない比較・冠詞・仮定法などの標準的な基礎事項も、初学者に必要な範囲で含めます。問題データの`basis`で、active原則に基づく問題と標準基礎事項を区別します。

2026-07-24に追加された助動詞の意味、助動詞＋`have p.p.`、冠詞・修飾・文型・自他動詞の判断の目安を、既存問題の改稿と出題根拠へ反映しました。助動詞は、形だけでなく許可・推量・義務・申し出・仮定法などの意味を確認します。

## ねらい

- 150問すべてを**知識｜補助**として扱い、英文法16分野の重要事項を一巡する
- 解答を選んだ時点で、正誤・正答・根拠を含む問題別解説をその場に表示する
- 誤答を「よくある混同」として結果に表示し、分野別の詳しい解説につなげる
- 通常利用ではブラウザの localStorage に保存し、生徒別共有URLでは共通Supabaseへ結果を同期する

## 結果の見方

- **5つの学習段階**: 品詞と文の骨組みから仮定法・語法までの到達を確認する
- **16分野**: 正答率から、知識が薄い分野を確認する

分野判定は、1問の誤答だけで「未定着」とはしません。正答率から「定着／要確認／未定着」とし、2問以下の分野は「要追加確認」と表示します。

問題は「品詞と文の骨組み→動詞の形と時制→助動詞・受動態と準動詞→準動詞と文のつなぎ方→比較・仮定法と仕上げ」の5段階で、1回30問ずつ固定出題します。選択肢の順だけは毎回入れ替えます。すべて「一問＝一判断」とし、選択肢を選んだ時点で採点します。

学習導線は第1段階から順番に進む方式です。誤答だけの別復習、間隔復習、未完了段階への直接移動は行わず、各段階の結果から次の段階へ進みます。

## 起動

```powershell
cd "C:\Users\shtom\dev\english-grammar-trainer"
py -3 -m http.server 8097
```

`http://127.0.0.1:8097/grammar-knowledge-check/` を開く。

## 構成

- `data/questions.js`: 16分野・知識｜補助150問、誤概念タグ、分野別解説、出題根拠区分、active原則への`ruleRefs`
- `static/app.js`: 出題、採点、学習段階別・分野別・誤概念別の結果、解説、ローカル保存・生徒別クラウド同期
- `static/styles.css`: アイボリー地・影なし・キーボード操作前提の画面

## 検証

```powershell
node --check data/questions.js
node --check static/app.js
node scripts/check-data.js
node scripts/export-question-list.js
```

`check-data.js` は問題数・選択肢・正解に加え、全問が知識｜補助であること、誤概念タグ、active原則への`ruleRefs`、固定順、「一問＝一判断」を検査します。否定表現や選択肢長の偏りは警告として出力します。

`export-question-list.js` は、`問題一覧.md` を問題データから再生成します。

問題集合を更新した場合は`contentVersion`を更新し、旧問題のlocalStorage・共有進捗を新問題へ持ち越しません。
