# 英語演習サイト v2

文法事項ごとに、説明・暗記4択・空所補充4択を進めるフロント完結の試作版。

## 起動

```powershell
cd "C:\Users\shtom\OneDrive\ドキュメント\Obsidian Vault\英語演習サイト\v2"
py -3 -m http.server 8095
```

ブラウザで開く。

```text
http://127.0.0.1:8095/
```

## 進捗保存

初期版では `localStorage` に保存する。

- 説明: 確認済み表示のみ
- 暗記: 暗記4択を全問正解でクリア
- 演習: 空所補充4択を全問正解でクリア
- 文法事項: 暗記クリア + 演習クリアで完了

## 問題生成

文法事項データから問題バンクを事前生成する。

```powershell
py -3 tools\generate_grammar_bank.py
```

入力:

```text
data/grammar_items.json
```

出力:

```text
data/grammar_bank.json
```
