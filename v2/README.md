```powershell

py -3 -m http.server 8095
```

ブラウザで開く。

```text
http://127.0.0.1:8095/
```

### 中学英文法コースの細分化（大項目→文法事項）

- be動詞: 肯定文(am/are/is) / 否定文・疑問文 / 過去形(was/were)
- 一般動詞: 現在形 / 三単現の s / 否定文・疑問文(do・does)
- 疑問文: 疑問詞 what/who/which/whose / 疑問詞 when/where/why/how
- 現在進行形: 現在進行形(be+-ing) / 否定・疑問・過去進行形
- 過去形: 規則動詞(-ed) / 不規則動詞
- 未来表現: will / be going to
- 助動詞: can(能力) / 許可・依頼の can・may
- 名詞: 複数形 / a・an・the
- 代名詞: 主格・所有格・目的格 / 所有代名詞・this・that
- There is構文: There is/are / 否定・疑問・過去
- 不定詞: 名詞的用法 / 副詞的・形容詞的用法
- 動名詞: 動名詞(〜すること) / 動名詞と不定詞(enjoy -ing/want to)

### 高校・大学受験コースの細分化（大項目→文法事項）

代表1項目だった粗い単元を、入試の出題単位に合わせて細分化済み（全16単元）。

- 時制: 現在過去 / 進行形 / 現在完了(継続・経験・完了) / 過去完了 / 未来完了 / 現在完了進行形（既存8項目）
- 接続詞: 等位・相関 / 従属 / 名詞節(that・whether) / so that・so…that
- 仮定法: 仮定法過去 / 仮定法過去完了 / 未来の仮定(should・were to) / 慣用(I wish・as if・if省略)
- 助動詞: 基本助動詞 / 義務・習慣(have to・used to) / 過去形のニュアンス / 助動詞+have p.p.
- 冠詞: a/an / the / 無冠詞
- 名詞: 可算名詞 / 不可算名詞 / 集合名詞・判別
- 形容詞・副詞: 形容詞の用法 / 数量形容詞 / 副詞の位置 / まぎらわしい副詞
- 文型: 自動詞と他動詞 / 第1〜3文型 / 第4文型 / 第5文型(使役・知覚)
- 不定詞: 名詞的 / 形容詞的 / 副詞的 / 重要構文(意味上の主語・too to・be to)
- 分詞: 形容詞用法 / 感情動詞の分詞 / 分詞構文 / 分詞構文の応用
- 受動態: 基本 / 文型の受動態 / by以外の前置詞 / 特殊な受動態(say型)
- 比較: 原級・倍数 / 比較級・最上級 / 重要表現(the+比較級・実質最上級) / no more than(鯨構文)
- 関係詞: 関係代名詞 / 関係副詞 / 関係代名詞 what / 複合関係詞
- 前置詞: 時 / 場所・方向 / 区別が必要な前置詞
- 否定・疑問: 否定表現 / notを使わない否定 / 疑問詞・間接疑問 / 付加疑問・感嘆文
- 強調構文・倒置: 強調構文 / 否定語倒置 / その他の倒置(so・neither)

これらの細分化項目は暗記4択を `memorySeeds` で明示執筆している（自動生成ではない）。

## 進捗保存

初期版では `localStorage` に保存する。

- 説明: 確認済み表示のみ
- 暗記: 暗記4択を全問正解でクリア
- 演習: 空所補充4択を全問正解でクリア
- 文法事項: 暗記クリア + 演習クリアで完了
