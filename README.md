# 英文法トレーナー

初学者向けの基礎知識チェックから、入試に向けたポラリス演習までをつなぐ静的英文法アプリのリポジトリです。

- 入試基礎演習: `ポラリス英文法ファイナル演習1`
- 基礎知識チェック: `grammar-knowledge-check`
- 英文解釈導入: `reading`

公開版のポラリス入口から、基礎知識チェックと英文解釈導入へ移動できます。

## 起動・確認

公開版やローカル版の入口はポータルで管理します。

```powershell
py C:\Users\shtom\dev\portal\launcher.py
```

入口と基礎知識チェックのリンクを含めて確認する場合は、リポジトリ直下で静的サーバーを起動します。

```powershell
cd "C:\Users\shtom\dev\english-grammar-trainer"
py -3 -m http.server 8096
```

`http://127.0.0.1:8096/ポラリス英文法ファイナル演習1/` を開きます。

英文解釈導入を確認する場合は、次を開きます。

`http://127.0.0.1:8096/reading/`

## 公開とデータ

GitHub Pages公開対象の静的ファイルと、開発中のローカルデータを分けて扱います。生徒URLや外部サービス設定を変更するときは、ポータル、QR、設定例も同時に確認します。

公開サイトでは英文法、基礎知識チェック、英文解釈導入を同じサイト内で提供します。各アプリの進捗保存領域とSupabaseの`appId`は分離しています。
