# 英文法トレーナー

初学者向けの基礎知識チェックから、入試に向けたポラリス演習までをつなぐ静的英文法アプリのリポジトリです。

- 入試基礎演習: `ポラリス英文法ファイナル演習1`
- 基礎知識チェック: `grammar-knowledge-check`

公開版のポラリス入口から、基礎知識チェックへ移動できます。

## 起動・確認

公開版やローカル版の入口はポータルで管理します。

```powershell
py C:\Users\shtom\dev\portal\launcher.py
```

単体で確認する場合は、次のフォルダで静的サーバーを起動します。

```powershell
cd "C:\Users\shtom\dev\polaris-grammar-final-1\ポラリス英文法ファイナル演習1"
py -3 -m http.server 8096
```

## 公開とデータ

GitHub Pages公開対象の静的ファイルと、開発中のローカルデータを分けて扱います。生徒URLや外部サービス設定を変更するときは、ポータル、QR、設定例も同時に確認します。
